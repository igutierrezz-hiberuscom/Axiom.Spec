# Plantilla de `context/README.md`

## Propósito

Esta carpeta contiene material añadido de apoyo a la spec general.

## Qué puede vivir aquí

- manuales de usuario;
- capturas;
- notas o transcripciones de reuniones;
- anexos;
- documentos adicionales no canónicos.

## Qué no debe vivir aquí

- conocimiento técnico estable de largo plazo, que pertenece a `technical-context/`;
- fuente de verdad estructural de incrementos, bugs o planes, que pertenece a metadata.

## Estructura sugerida

- `manuals/`
- `screenshots/`
- `meeting-notes/`
- `annexes/`
